Welcome to jqnode

jqnode is a lightweight jQuery-compatible library for DOM manipulation and traversal. It works seamlessly in both Node.js environments and web browsers, making it ideal for server-side HTML processing, web scraping, and client-side DOM manipulation.

The core of jqnode is the $() factory function. It can parse HTML strings, select elements with CSS selectors, or work with existing node arrays.
